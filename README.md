# 🔐 SecureCloud

SecureCloud is a cloud-based web application that provides secure authentication, data encryption, and real-time threat detection.

## 🚀 Features

* 🔐 Secure Login & Registration (bcrypt hashing)
* 🛡️ SQL Injection Detection
* 🔒 AES Data Encryption
* ☁️ Cloud Storage (Firebase Firestore)
* 📊 Admin Dashboard with Attack Visualization

## 🛠️ Technologies Used

* Python (Flask)
* Firebase Firestore
* HTML, CSS, Bootstrap
* Chart.js
* bcrypt & AES Encryption

## 📂 Modules

* Authentication System
* Attack Detection Engine
* Encryption System
* Cloud Database Integration
* Admin Monitoring Dashboard

## ▶️ How to Run

```bash
pip install flask firebase-admin bcrypt cryptography
python app.py
```

Open in browser:
http://127.0.0.1:5000
